🚀 Getting Started

Prerequisites

  • Node.js v18 or higher
  • npm v9 or higher
  • Supabase account (free tier)
  • Stripe account (for payments)

Installation

1

Clone the repository

git clone <repository-url>

2

Install dependencies

npm install

3

Configure environment variables

Create .env.local with your Supabase and Stripe keys

4

Run the development server

npm run dev

Environment Variables

# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=sb_publishable_xxxxx
SUPABASE_SERVICE_ROLE_KEY=sb_secret_xxxxx

# JWT
JWT_SECRET=your-jwt-secret-64-chars
JWT_REFRESH_SECRET=your-refresh-secret-64-chars

# Stripe
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_xxxxx
STRIPE_SECRET_KEY=sk_test_xxxxx

🔐 Authentication

User Flows

  • Register: /register — Create new account with username, email, password
  • Login: /login — Sign in with email and password
  • Forgot Password: /forgot-password — Receive reset code via email
  • Reset Password: /reset-password — Enter code and set new password
  • Email Confirmation: Automatic email sent after registration

📦 Products

Product Management (Admin)

  • Add products with name, description, price, category, image
  • Edit existing products inline
  • Delete products with confirmation
  • Set featured status for homepage display
  • Manage stock quantity

Categories

Default categories: ZARA NIKE DOM HILL WOMEN ACCESSORIES

📋 Orders

Order Status Flow

Pending → Processing → Shipped → Delivered

Status Meanings

  • Pending — Order placed, awaiting payment confirmation
  • Processing — Payment confirmed, preparing order
  • Shipped — Order dispatched to customer
  • Delivered — Order received by customer
  • Cancelled — Order cancelled

💳 Payment

Stripe Integration

The platform uses Stripe Checkout for secure payment processing.

Test Mode

Use test card: 4242 4242 4242 4242 with any future date and any 3-digit CVC.

Live Mode

  1. Activate your Stripe account
  2. Get your Live Keys from Stripe Dashboard
  3. Replace pk_test_* with pk_live_*
  4. Replace sk_test_* with sk_live_*
